새 블로그 포스트 파일을 생성해줘.

**입력값 파싱**
사용자가 제공한 인자($ARGUMENTS)에서 다음을 추출해:
- 제목 (title)
- 태그 (tags, 쉼표로 구분 / 없으면 빈 배열)

**slug 규칙**
제목을 kebab-case로 변환:
- 한글은 그대로 유지 (예: "Astro 시작하기" → "astro-시작하기")
- 영문은 소문자 변환
- 공백 → 하이픈(-)
- 특수문자 제거

**생성할 파일**
`src/content/blog/[slug].md`

**파일 내용 형식**
```
---
title: "[제목]"
description: ""
pubDate: [오늘 날짜, YYYY-MM-DD 형식]
tags: [[태그 배열]]
---

## 들어가며

<!-- 여기에 내용을 작성하세요 -->

## 본문

## 마치며
```

**완료 후 출력**
- 생성된 파일 경로
- 접근 URL 예시: `/blog/[slug]`
- description 작성 안내 메시지
